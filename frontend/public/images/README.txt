Carpeta de imágenes del frontend

Sube aquí el logo de tu estudio para que aparezca en la pantalla de login:
  - logo.png   (recomendado: PNG con fondo transparente, aprox. 200–400 px de ancho)
  - logo.svg   (también válido; en el código se usa logo.png por defecto)

Si no hay imagen, se mostrará el texto "NT" como respaldo.
